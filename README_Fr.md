# GLPI

## Présentation

GLPI (Gestionnaire Libre de Parc Informatique) est une plateforme open source bien connue de gestion des services informatiques (ITSM).

Elle fournit une interface web unique permettant de gérer les actifs informatiques, les demandes d'assistance, les incidents, les tickets, les workflows et la documentation technique. GLPI peut être utilisé comme helpdesk, système d'inventaire et CMDB (Configuration Management Database) pour le suivi des postes de travail, serveurs, logiciels, licences, équipements réseau et informations de cycle de vie.

Sur notre site, GLPI est principalement utilisé pour la gestion de l'inventaire du parc informatique, compte tenu du nombre relativement limité de postes et d'équipements réseau présents sur site.

## Fonctionnalités principales

* Inventaire des équipements informatiques
* Suivi du cycle de vie des actifs
* Gestion des incidents, demandes et tickets
* Fonctions de helpdesk et de support utilisateur
* CMDB pour documenter les relations entre les ressources et les services
* Gestion des logiciels et des licences
* Base de connaissances et documentation technique
* Administration centralisée via une interface web

## Pourquoi GLPI ?

GLPI permet de disposer d'une vue centralisée et structurée de l'environnement informatique.

Exemples d'utilisation :

* Inventorier les ordinateurs, imprimantes et équipements réseau
* Suivre les logiciels installés et les licences
* Documenter l'infrastructure informatique
* Gérer les demandes d'assistance et les incidents
* Maintenir un inventaire à jour des actifs informatiques

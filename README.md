# GLPI

## Overview

GLPI (Gestionnaire Libre de Parc Informatique) is a well-known free and open-source IT Service Management (ITSM) platform.

It provides a unified web interface for managing IT assets, service requests, incidents, tickets, workflows and technical documentation. GLPI can be used as a helpdesk, an asset inventory system and a CMDB (Configuration Management Database) for tracking computers, servers, software, licenses, network devices and lifecycle information.

On our site, GLPI is primarily used for inventory management because of the relatively small number of computers and network devices on site.

## Key Features

* IT asset inventory and lifecycle tracking
* Incident, request and ticket management
* Service desk and helpdesk functions
* CMDB for documenting relationships between resources and services
* Software and license management
* Knowledge base and technical documentation
* Centralized web administration

## Why GLPI?

GLPI provides a structured and centralized view of the IT environment.

Typical use cases include:

* Keeping track of computers, printers and network equipment
* Recording software installations and licenses
* Documenting infrastructure components
* Managing support requests and incidents
* Maintaining an up-to-date inventory of IT assets
